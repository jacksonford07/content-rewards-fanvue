### D3. Domain decision = setup-only

Currently `content-rewards.fanvue.call.app`. Possible upgrades:
- Subdomain on a Fanvue-owned domain: `rewards.fanvue.com` (cheapest; aligns with brand).
- Standalone: `contentrewards.app` or similar (more independent; possibly slower trust to land in users' minds as a Fanvue product).

**Council recommendation: subdomain on `fanvue.com`.** Rewards is a Fanvue product, not an independent service. Independent domain spends brand equity for no clear benefit. Iniaki + Jackson confirm + buy if needed.

This is purely a Vercel domain config — no code change.
